HIVABM:  Agent-based modeling of drug use and HIV spread
=============================================================

`HIVABM` is a python module providing all classes and
functions for an agent-based model of drug use and HIV spread. 

.. note::

   This documentation assumes general familiarity with Object-Oriented
   Programming in Python. For an introduction to Object-Oriented Programming in Python see, e.g., `here <http://docs.python.org/tutorial/classes.html>`__ .  The module requires `NumPy <http://docs.scipy.org>`__ , `Matplotlib <http://matplotlib.sourceforge.net/>`__  and `NetworkX <http://networkx.lanl.gov/>`__ .
